建立about_item資料表

前面的list只是about頁面的框架，內文由type分類
再用item各別裝進去

這樣看來資料庫系統其實比較像框架，將各種資料分類
像一個大抽屜，裡面放很多格子
要調用的時候可以自動化，不用再一個一個手動連結

  1. 在about_type資料表裡新增items欄位
    由於是布林，因此值只要設1（Y/N）

    名稱 類型 長度/值 Null 備註
    items char 1 O 有無項目

  2. 將about_type裡的「關於我們」、「優勢」兩欄位的items設為Y

  3. 在app\Models\About\AboutType.php裡$fillable新增項目items

  4. 建立新資料表about_items

    名稱 資料類型 長度/值 Null A.I 備註
    id int - - O -
    typeId int - O - -
    itemName varchar 50 O - -
    content varchar 500 O - -
    photo varchar 30 O - -
