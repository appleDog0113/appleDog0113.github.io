##### CSS效果設定

> 1. 在`.box_in`上面覆蓋一層`.roller`作為卷軸
> 2. `.box_in`裡設定`.roller`的css動畫效果

```css
.content .box {
  display: inline-block;
  width: 32%;
  height: auto;
  margin-top: 15px;
  .box_in {
    position: relative;   /* 相對位置 */
    height: auto;
    .roller {
      display: flex;
      justify-content: center;
      align-items: center;
      /*-- 終究要用flex--*/
      position: absolute;   /* 絕對位置 */
      width: 100%;
      height: 100%;
      background-color: var(--mycolor05);
      opacity: 0%;
      top: -30%;
    }
  }
  .box_in:hover .roller {
      opacity: 100%;
      transition: 750ms;
      top: 0%;
  }
}
```
